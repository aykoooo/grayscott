## 📐 Key Technical Details

- **Shader is runtime-generated** — inside `generateWgsl()` in `src/gpu/gpu.zig`. The `.wgsl` file is NOT used by the benchmark.
- **Workgroup size** is currently `16x16` (256 threads). Try other sizes: `8x8`, `32x4`, `16x8`, `32x32`, `64x2`.
- **Shared memory in WGSL**: Declare as `var<workgroup> tile: array<f32, 20*20>;` for a 16x16 tile with 2-cell halo. Use `workgroupBarrier()` to synchronize.
- **f16 feature**: Check `shader-f16` WebGPU support. Use `enable f16;` at shader top. Convert f16↔f32 only for math.
- **Ping-pong buffers**: U0→U1 step 1, U1→U0 step 2. Two bind groups swap.
- **Bounds handling**: Currently uses `select()` for periodic. Could try clamped for map mode, periodic for standalone.
