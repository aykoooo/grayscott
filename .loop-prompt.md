## 📐 Key Technical Details

- **Shader is runtime-generated** — inside `generateWgsl()` in `src/gpu/gpu.zig`
- **Current workgroup**: 8×8 (64 threads) — `wg_x`/`wg_y` in GpuState
- **Shared memory tiling IS DONE**: loads (TX+2)×(TY+2) tile into `var<workgroup>` arrays with edge/corner halo
- **Command buffer batching IS DONE**: `gs_gpu_steps(N)` records N dispatches in one command encoder, submits once
- **Ping-pong buffers**: U0→U1 step 1, U1→U0 step 2. Two bind groups (even/odd) swap based on `current_u`
- **Bounds handling**: `select()` for periodic boundaries in both tile loading and computation
- **f16 support**: `has_f16` flag detected via `wgpuAdapterHasFeature()`, but f16 storage was reverted (compute-bound, no gain)
- **Benchmark hash**: `e16ed0e3c29cc50b5fa2b42791f31ab00b39d488e971b5d3c6017970ed037a43` — sacred, must always match
